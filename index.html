المقدمة والأهمية

مشروع Lesan Blur هو إضافة (Extension) لمتصفح جوجل كروم تهدف إلى جعل بيئة وسائل التواصل الاجتماعي أكثر أماناً ونظافة.

لماذا قمنا ببناء هذه الإضافة؟

تنتشر الكلمات المسيئة والشتائم بشكل كبير في التعليقات والمنشورات على وسائل التواصل الاجتماعي، مما قد يؤثر سلباً على تجربة المستخدمين ويخلق بيئة غير صحية خاصة للشباب والأطفال. جاءت فكرة Lesan Blur لتقوم بحجب هذه الكلمات بشكل تلقائي بمجرد ظهورها على الشاشة.

أهمية المشروع

  • حماية المجتمع: تقليل التعرض للمحتوى اللفظي المسيء.
  • تجربة تصفح هادئة: التركيز على المحتوى المفيد دون تشتت بسبب التعليقات السلبية.
  • الخصوصية وعدم التتبع: الإضافة تعمل محلياً (Local) في متصفحك وتقوم بمسح النصوص لحظياً باستخدام تقنيات DOM، بدون إرسال بياناتك لأي خوادم خارجية.

Introduction & Importance

The Lesan Blur project is a Google Chrome Extension aimed at making social media environments safer and cleaner.

Why did we build this extension?

Offensive words and profanity are widespread in comments and posts on social media, which can negatively impact the user experience and create an unhealthy environment, especially for youth and children. Lesan Blur was created to automatically blur these words as soon as they appear on the screen.

Importance of the Project

  • Community Protection: Reducing exposure to offensive verbal content.
  • Peaceful Browsing Experience: Focusing on useful content without being distracted by negative comments.
  • Privacy and No Tracking: The extension operates locally in your browser and scans text instantly using DOM techniques, without sending your data to any external servers.

دليل الاستخدام

استخدام إضافة Lesan Blur بسيط جداً ومصمم ليكون يعمل في الخلفية دون إزعاج.

كيفية التثبيت (للمطورين حالياً)

  1. قم بتحميل مجلد المشروع Lesan blur.
  2. افتح متصفح جوجل كروم واذهب إلى chrome://extensions/.
  3. قم بتفعيل خيار Developer mode في أعلى اليمين.
  4. اضغط على Load unpacked واختر مجلد المشروع.

كيفية التشغيل والإيقاف

بمجرد تثبيت الإضافة ستظهر أيقونتها في شريط إضافات المتصفح. عند الضغط عليها ستفتح واجهة (Popup) تحتوي على زر تشغيل (Toggle).

  • التشغيل (ON): سيتم مراقبة المواقع المدعومة وحجب أي كلمات مسيئة تظهر.
  • الإيقاف (OFF): ستتوقف الإضافة عن العمل، وستظهر جميع النصوص بشكلها الطبيعي.

Usage Guide

Using the Lesan Blur extension is very simple and designed to work seamlessly in the background.

How to Install (For Developers Currently)

  1. Download the project folder Lesan blur.
  2. Open Google Chrome and navigate to chrome://extensions/.
  3. Enable the Developer mode option in the top right corner.
  4. Click on Load unpacked and select the project folder.

How to Toggle ON/OFF

Once installed, the extension icon will appear in the browser's extensions bar. Clicking it opens a Popup interface with a toggle switch.

  • ON: Supported sites will be monitored, and any offensive words will be blurred out.
  • OFF: The extension will stop working, and all text will appear normally.

إعدادات الإضافة (Manifest)

ملف manifest.json هو القلب النابض للإضافة، فهو يخبر المتصفح عن ماهية الإضافة وصلاحياتها.

الصلاحيات (Permissions)

نستخدم صلاحيات مثل storage لحفظ حالة الإضافة (مفعلة/معطلة)، و activeTab للتعامل مع التبويبة الحالية.

النطاقات المدعومة (Host Permissions)

تم تحديد المواقع التي يحق للإضافة العمل عليها، مثل الفيسبوك وتويتر وجوجل.

Extension Configuration (Manifest)

The manifest.json file is the beating heart of the extension. It tells the browser what the extension is and its permissions.

Permissions

We use permissions like storage to save the extension's state (enabled/disabled), and activeTab to interact with the current tab.

Host Permissions

We specify which websites the extension is allowed to operate on, such as Facebook, Twitter, and Google.

"host_permissions": [
  "*://*.facebook.com/*",
  "*://*.youtube.com/*",
  "*://*.instagram.com/*",
  "*://*.twitter.com/*",
  "*://*.x.com/*",
  "*://*.google.com/*"
]

منطق الحجب (Content Scripts)

1. قاموس الكلمات (words.js)

يحتوي على مصفوفة badWords والتي تضم الكلمات المسيئة باللغتين العربية والإنجليزية. يتم تحويلها إلى تعبير نمطي للبحث.

2. محرك الحجب (content.js)

يستخدم TreeWalker للبحث عن النصوص ومطابقتها مع القاموس. وفي حال وجود كلمة مسيئة، يتم حجبها بـ <span>. كما يعتمد على MutationObserver لمراقبة أي محتوى جديد يظهر عند التمرير.

3. تصميم الحجب (content.css)

Blur Logic (Content Scripts)

1. Word Dictionary (words.js)

Contains the badWords array which includes offensive words in both Arabic and English. This is converted into a regular expression for searching.

2. Blur Engine (content.js)

Uses a TreeWalker to find text nodes and match them against the dictionary. If an offensive word is found, it is wrapped in a blurred <span>. It also relies on a MutationObserver to monitor dynamically loaded content during scrolling.

3. Blur Styling (content.css)

.lesan-blurred-text {
  filter: blur(5px);
  background-color: #333;
  color: transparent;
  pointer-events: none;
}

العمليات في الخلفية (Background)

ملف background.js يعمل كـ Service Worker لمعالجة التحديثات وحفظ حالة التشغيل عبر chrome.storage.sync والإستماع للرسائل القادمة من الـ Popup.

Background Tasks

The background.js file acts as a Service Worker to process updates, save the toggle state via chrome.storage.sync, and listen for messages coming from the Popup UI.

واجهة المستخدم (Popup)

الواجهة التي تظهر للمستخدم عند الضغط على أيقونة الإضافة بتصميم داكن احترافي، وتحتوي على زر تشغيل بسيط يقوم بإرسال الأوامر لتحديث الحجب فوراً في كل الصفحات النشطة.

User Interface (Popup)

The interface that appears when the user clicks the extension icon. Designed with a premium dark mode, it features a simple toggle switch that sends commands to update the blurring immediately across all active tabs.

What's New? (Changelog)

v2.0.0

Second Release — Full Protection Update 🛡️

A major update that adds multiple advanced protection layers to provide a completely safe browsing environment.

New

⛔ Incognito Mode Blocker

Added a critical new security feature that completely prevents users from accessing Incognito Mode. When a user attempts to open an incognito window, the extension automatically displays a premium-designed warning page explaining that this mode is blocked for safety reasons, then closes the window automatically within 5 seconds. This feature prevents users (especially children and young people) from bypassing the protection system through private browsing.

New

🔄 Smart Redirection for Social Media Search

The smart search prevention feature has been expanded to cover all major social media platforms (YouTube, Facebook, Twitter X, Instagram, TikTok, etc.). If a user attempts to search for a bad word on any of these platforms, the extension intercepts the search query instantly and automatically redirects the browser to search for safe and positive content (e.g., "funny and positive content").

New

🌐 Added Support for Google Search

The Google search engine has been officially integrated into the list of supported domains. The extension is now capable of scanning Search Engine Results Pages (SERP) and proactively blurring any offensive words in search titles or snippets before you even click a link, providing an excellent layer of preemptive protection.

Update

📖 Comprehensive Bilingual Documentation

Launched a fully independent and comprehensive documentation platform for the extension. It fully supports both English and Arabic languages, and features a modern, eye-friendly design based on the 60-30-10 color rule with an explicit Dark/Light mode toggle to help developers and users understand and contribute to the project.
